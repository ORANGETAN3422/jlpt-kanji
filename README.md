# JLPT Kanji

Simple website to make tracking progress on kanji you have learnt for the JLPT easy.

Check off completed kanji, view stroke definitions, readings and stroke orders, and look at example words.

Powered by [kanjiapi.dev](https://kanjiapi.dev/), [JLPT Vocabulary Api](https://jlpt-vocab-api.vercel.app/) and [Jisho](https://jisho.org)

Built with [Svelte](https://svelte.dev/), [SvelteKit](https://kit.svelte.dev/) and [tailwindcss](https://tailwindcss.com)
